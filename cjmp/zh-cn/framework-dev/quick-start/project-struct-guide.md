# 代码工程介绍


## 目录结构

```
项目根目录
├── build - 构建系统相关文件夹
├── ci - 持续集成相关文件夹
├── cj-ui - 用户界面相关文件夹
├── docs - 文档文件夹
├── Engine - 引擎源码文件夹
│   ├── build - 引擎的构建脚本文件夹
│   ├── foundation - 基础设施相关文件夹
│   ├── test - 测试文件夹
│   ├── utils - 公共库文件夹
│   ├── build.sh - 构建脚本的快捷方式
│   └── ...
├── examples - 示例代码文件夹
├── ide-plugins - IDE 插件文件夹
├── systemlibs - 逻辑相关文件夹
├── openharmony_prebuilts - OpenHarmony 预构建文件夹
├── platform - 平台相关代码文件夹
├── test - 项目测试文件夹
├── third_party - 第三方库文件夹
├── tools - 工具文件夹
└── build.sh - 集成编译脚本
```
